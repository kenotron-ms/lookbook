One command. Any stack. Any cloud.

GRID.SH turns your git push into a production deployment across any cloud — AWS, GCP, Fly.io, Hetzner — with zero configuration files, zero YAML, and zero platform lock-in. Your team writes code. We handle the rest.

# Install globally
curl -fsSL https://grid.sh/install | sh
 
# Initialize in your project
grid init --auto
 
# Deploy
grid up --prod
# Deployed to grid.sh/live/your-app in 14s
