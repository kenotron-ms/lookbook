Built for developers. Loved by everyone.

⌨

Universal Command Palette

⌘K to open. Search tabs, history, bookmarks, and custom commands. Fuzzy matching, ranked by recency.

⌘K to activate
📦

Session Manager

Save your current tab state as a named session. Restore in one click. Perfect for context switching.

tf session save
🗂

Smart Tab Groups

Auto-group tabs by domain, topic, or custom rules. Keep your browser organized without thinking.

tf group --auto
🔗

Tab Link Sharing

Share a session or group of tabs as a single shareable link. Perfect for team handoffs.

tf share --session
🔍

Full-Text Search

Search the content of all open tabs in real time. Find that article you opened 2 hours ago instantly.

tf search --fulltext
⚡

Custom Shortcuts

Assign any action to any keyboard shortcut. 40+ built-in actions with a scriptable API for custom ones.

tf shortcut bind
