Tools

Everything a dark-first designer needs. Nothing extra.

Contrast Analyzer

Real-time WCAG 2.2 contrast checking as you design. Fail states surface immediately — before they reach code review.

Token Editor

Edit semantic tokens with live preview across your entire component library. Changes cascade — no manual find-and-replace.

Spotlight View

Code Sync
